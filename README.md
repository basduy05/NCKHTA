# EAM Project - NCKHTA 2025 (Advanced AI Ecosystem)

Welcome! This is a cutting-edge educational platform built for the **Scientific Research Competition (NCKHTA 2025)**. It leverages **Knowledge Graphs (Neo4j)** and **Generative AI (LLMs)** to create a personalized, adaptive English learning experience.

## 🌟 Key Innovations (Breakthrough Features)
1. **Knowledge Graph Visualization:** See your brain's network grow as you learn new words.
2. **AI Flashcard Generator:** Instantly create learning materials from any text using GPT-4/Gemini.
3. **GraphRAG Chatbot:** An AI tutor that understands context and relationships between concepts, not just keywords.
4. **Real-time Pronunciation Feedback:** (Planned) Analyze speech patterns for correction.

## 🚀 Step-by-Step Deployment (Updated)


### 1. Database Setup (Do this first!)
**PostgreSQL (Relational DB):**
- **Option A (Easy):** Use the `render.yaml` blueprint below, which creates a Postgres DB on Render automatically.
- **Option B (Alternative):** Create a project on [Supabase](https://supabase.com/). Get the connection string. USE the script in `database/schema.sql` to create the tables.

**Neo4j (Graph DB):**
1. Go to [Neo4j AuraDB](https://neo4j.com/cloud/aura/) and create a free instance.
2. Save the `Connect URI`, `Username`, and `Password`.
3. Use the script in `database/graph_schema.cypher` to initialize constraints.

### 2. Backend Deployment (Render)
1. Push this entire folder (`NCKHTA`) to a new **GitHub Repository**.
2. Go to [Render Dashboard](https://dashboard.render.com/).
3. Click **New +** -> **Blueprint**.
4. Connect to your GitHub repo.
5. Render will detect `render.yaml`. Click **Apply**.
   - It will ask for environment variables like `NEO4J_URI`, `NEO4J_PASSWORD`. Enter the values you got from Step 1.
   - It will setup Postgres, Spring Boot, and FastAPI automatically.

### 3. Frontend Deployment (Vercel)
1. Go to `frontend/` folder locally.
2. Run `npm install` then `npm run build` to ensure it works.
3. Push code to GitHub.
4. Go to [Vercel](https://vercel.com/new).
5. Import the same GitHub repo.
6. **Important:** In "Root Directory", click Edit and select `frontend`.
7. Click **Deploy**.

## 🛠 Local Development
To run everything locally, you can use the individual commands or Docker (advanced).

**Requirements:**
- Java 17+ & Maven
- Python 3.9+
- Node.js 18+

**Commands:**
- **Backend:** `cd backend-core && mvn spring-boot:run`
- **AI Service:** `cd ai-service && uvicorn app.main:app --reload`
- **Frontend:** `cd frontend && npm run dev`
